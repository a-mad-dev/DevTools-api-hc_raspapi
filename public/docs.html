Devtools API

Simple utility API for developers

Base URL

https://devtools-api-hc-raspapi.onrender.com/

GET /random

Returns a random number (0–99)

{ "number": 42 }

GET /uuid

Returns a random UUID

{ "uuid": "..." }

GET /hash?text=hello

Returns SHA256 hash of text

{ "text": "hello", "hash": "..." }

POST /json/format

Wrap your JSON inside { data: {} }

Request:
{
  "data": {
    "a": 1
  }
}

Response:
{
  "valid": true,
  "formatted_json": { "a": 1 }
}
